About

Hello, My name is Lora and I‘m a graphic designer. So far I‘ve been able to work on various projects, creating brands, visuals, prints and many more. I‘ve had the opportunity to work for world-wide companies like Fujitsu, adhering strict visual standards, while at the same time I enjoyed projects where I had freedom to create and experiment. Willing to expand my horizon, last year I enrolled the SoftUni program for JavaScript developers, since web was something I always considered an immense field for design and creation. To my surprise, coding turned out not only to be a lot of fun, but also quite interesting and provocative. I dedicated all my time to learning new technologies and frameworks. Although I‘m still a beginner, I‘m pretty sure this is something I want to explore deeper. I found a great tool to experiment with and a huge challenge to become better at. I‘m really interested in creating impressive, yet simple and clean design, combining good structure, consistency and usability.
